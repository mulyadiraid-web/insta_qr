# QR Generator - Premium Edition 🚀

A sleek, modern, and high-performance QR Code generator built with Flutter. Convert any link or text into a professional QR code with just one click.

![Premium Design](https://img.shields.io/badge/Design-Premium-blueviolet?style=for-the-badge)
![Flutter](https://img.shields.io/badge/Flutter-v3.22+-02569B?style=for-the-badge&logo=flutter&logoColor=white)

## ✨ Features

- **Instant Generation**: Real-time QR code generation for links, emails, or plain text.
- **Premium UI**: Stunning dark mode aesthetic with vibrant gradients and glassmorphism.
- **Animations**: Smooth micro-animations for a dynamic user experience.
- **Save & Share**: Easily save your QR codes to the gallery or share them directly with friends.
- **Responsive**: Optimized for both Android and iOS devices.

## 📦 Dependencies

- [qr_flutter](https://pub.dev/packages/qr_flutter) - For generating QR codes.
- [google_fonts](https://pub.dev/packages/google_fonts) - Premium typography.
- [animate_do](https://pub.dev/packages/animate_do) - Smooth animations.
- [screenshot](https://pub.dev/packages/screenshot) - Capturing QR images.
- [share_plus](https://pub.dev/packages/share_plus) - Native sharing capabilities.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/insta_qris.git
   ```
2. **Install dependencies**:
   ```bash
   flutter pub get
   ```
3. **Run the app**:
   ```bash
   flutter run
   ```

## 🎨 UI Concept

The app uses a deep space-themed gradient (`#0F2027` → `#203A43` → `#2C5364`) combined with vibrant Cyan accents (`#00B4DB`) to create a futuristic and premium feel.
